about

The developer behind the code

A closer look at how I work and what I bring to your project.

I'm Mark Opiyo, a freelance fullstack developer based in Nairobi, working with clients across web development, software development, and website maintenance. I care about building things that are fast, reliable, and genuinely useful — not just software that looks good in a demo.

My work spans the full stack: from designing the interface someone actually touches, to the backend logic and databases that keep it running, to keeping a site healthy long after launch. I also specialize in integrating AI tools directly into web apps — think smart search, automation, chat assistants, and content generation — built to solve a real problem, not bolted on for buzzwords.

I work closely with each client, communicate clearly, and treat every project like it's going into production — because it usually is.
